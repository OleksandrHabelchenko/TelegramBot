# Telegram Bot

Try it: [@amazing_first_bot](https://t.me/amazing_first_bot)

Telegram bot built with Python.

## Technologies
- Python 3
- pyTelegramBotAPI
- OpenWeatherMap API

## Features
- Weather by city
- Random jokes
- Current time
- Inline and reply keyboards
- About section with links

## How to run
1. Install dependencies: pip install pytelegrambotapi requests
2. Add your token in bot.py
3. Run: python bot.py
